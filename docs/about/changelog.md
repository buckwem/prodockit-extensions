# Release Notes

## 0.1.0 (unreleased)

- Initial `zendoc.headings` extension: heading ids and hierarchical section
  numbering, backed by a shared `IdRegistry`.
- Initial `zendoc.refs` extension: `\ref{id}` section cross-references,
  resolving to the target's current section number, including forward
  references within a document and across a shared registry.
- Not yet released to PyPI.
